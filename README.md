Öğrenci Bilgi Sistemi

Bu proje; C# ve SQL Server kullanılarak geliştirilmiş bir Windows masaüstü uygulamasıdır.
Öğrenci, akademisyen ve not bilgilerini yönetmeye yarayan, şifre korumalı bir Öğrenci İşleri Bilgi Sistemidir. 
Kullanıcılar sisteme giriş yaparak öğrenci kayıtlarını, akademisyen bilgilerini ve not işlemlerini takip edip yönetebilir.
